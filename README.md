# congaaas
Conga as a Service

Converts a gif to a conga-ized gif. Looping gifs only please!

## To install:
```
git clone https://github.com/hschallh/congaaas
cd congaaas
chmod +x script
```


## To do the thing
```
cd <congaaas_directory>
mv <location_of_gif_to_conga> ./original.gif
./script
```

congaaa'd gif is true.gif
